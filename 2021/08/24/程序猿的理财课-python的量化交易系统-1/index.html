获取所有股票行情数据

课程中的数据来源是用一个名字叫聚宽的量化平台来获取，注意的是这个量化平台的数据调用有限制 还有新注册的用户只有半年的时间 如果可以的话 还是从别地渠道获取数据 ，或者自己爬虫写一个 都是些api文档 小白都能看得懂 这里就不做笔记了

转换周期函数resample

resample函数是一个pandas的一个函数 通过它 可以把一组由时间为索引的数据进行采样 可以通过传递下面图片的参数来获取你想要的数据 比如 我想要每周的第一天的开盘价 我就传递‘W ’参数 筛选出以周为间隔的数据 然后在对数据处理 比如我想要每周一的数据 可以通过first（）方法来获取
参数 说明
freq 表示重采样频率，例如‘M’、‘5min’，Second(15)
how=’mean’ 用于产生聚合值的函数名或数组函数，例如‘mean’、‘ohlc’、np.max等，默认是‘mean’，其他常用的值由：‘first’、‘last’、‘median’、‘max’、‘min’
axis=0 默认是纵轴，横轴设置axis=1
fill_method = None 升采样时如何插值，比如‘ffill’、‘bfill’等
closed = ‘right’ 在降采样时，各时间段的哪一段是闭合的，‘right’或‘left’，默认‘right’
label= ‘right’ 在降采样时，如何设置聚合值的标签，例如，9：30-9：35会被标记成9：30还是9：35,默认9：35
loffset = None 面元标签的时间校正值，比如‘-1s’或Second(-1)用于将聚合标签调早1秒
limit=None 在向前或向后填充时，允许填充的最大时期数
kind = None 聚合到时期（‘period’）或时间戳（‘timestamp’），默认聚合到时间序列的索引类型
convention = None 当重采样时期时，将低频率转换到高频率所采用的约定（start或end）。默认‘end’


通过resample函数可以把数据转换为周k 月k 等

df_week = pd.DataFrame()
df_week['open'] = df['open'].resample('W').first() #一周的开盘价
df_week['close'] = df['close'].resample('W').last()# 一周的收盘价
df_week['high'] = df['high'].resample('W').max() # 一周的最高价
df_week['low'] = df['low'].resample('W').min() # 一周的最低价

还可以统计一个月的成交额

df_week['volume(sum)'] = df['volume'].resample('M').sum()
