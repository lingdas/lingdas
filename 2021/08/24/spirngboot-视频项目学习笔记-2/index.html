vue的使用

安装vue 

npm install -g @vue/cli

创建一个新项目admin 会生成以下目录

vue create admin


package.json 类似于我们项目中 pom.xml 管理js的各种依赖
public/index.html 启动页面

<!DOCTYPE html>
<html lang="">
  <head>
    <meta charset="utf-8">

  </head>

    </noscript>

    <!-- built files will be auto injected -->
  </body>
</html>

src/main.js 启动脚本

import Vue from 'vue'
import App from './App.vue'

Vue.config.productionTip = false

new Vue({
  render: h => h(App),
}).$mount('#app')

其中new Vue({ render: h => h(App),}).$mount(‘#app’) 把App.vue 文件的内容渲染到 index.htmlid名为app的div中
App.vue 文件 主要有三个部分组成 template （html结构） script（js脚本） style（样式）


<template>
  <div id="app">
    <img alt="Vue logo" src="./assets/logo.png">
    <HelloWorld msg="Welcome to Your Vue.js App"/>
  </div>
</template>

<script>
import HelloWorld from './components/HelloWorld.vue'

export default {
  name: 'App',
  components: {
    HelloWorld
  }
}
</script>

<style>
#app {
  font-family: Avenir, Helvetica, Arial, sans-serif;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
  text-align: center;
  color: #2c3e50;
  margin-top: 60px;
}
</style>
